# bespoke-nails
Bespoke Nails Website
  
  
  I have designed this website for my project called Bespoke Nails which is a small independent nail salon. I chose to do a nail salon website because I am actually a qualified nail tech so I thought this would a good idea to do. The website allows users  to view information about the salon's services, prices and how to book an appointment. I designed it to be responsive to mobile and desktop devices.


The aim of my project
  
  
 Is to design and develop an easy to navigate and functional website for business and its users.


The Key Features of the website
  
  
  With this website you will be able to view nail services and prices.
  Book an appointment online.
  Navigate easily to different sections.
  See salon's contact details and location.


User Requirements
   
   
  I designed the website based on the user stories and prioritised them by making sure the essential features were completed first.


I created my wireframes at the beginning of my project, to plan the layout and structure of my website. This helped to plan the content, placement and navigation before applying design and writing codes. As you will see I descided to incorporate the About section on to my homepage

    Tinypng to convert images to webp

    I reduced the image sizes in the very beginning but when I tested the website through Lighthouse it recommended I reduce even more and convert png to webp or avif files. So I deleted all my images from the gallery and resized and converted to webp.

    I used color palettes light brown #aa9581 and light pink #febcd4. I chose to use those colors to match the theme of the images but unfortunately when I tested it through Lighthouse it failed the color ratio. So I checked the colors in contrast checker and they definitely failed. So I adjusted the color to a dark brown #463a2f text on light pink background which then passed. The white background and light pink text still failed, this is a mistake on my part because I completely forgot to do a color ratio first. So unfortunately I couldn't really change the light pink without changing the whole theme color of the website which I didn't want to do. So I know for sure to check color ratio first.
    I also chose 'Dancing Script' and 'Philosopher' fonts because I felt it was more visually appealing to use for a salon.
